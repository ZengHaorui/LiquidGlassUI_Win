# LiquidCpp UI Library

**A lightweight Liquid Glass C++ UI library for windows based on Imgui**

---

## Features

- **Minimalistic Design**: Focused on simplicity and ease of use.
- **Customizable**: Easily extendable to fit your needs.

---

## Getting Started

### Build Instructions

1. Ensure you have the following tools installed:
   - Visual Studio 2022 or later
   - CMake
   - Git

2. Clone the repository:
   ```bash
   git clone <repository-url>
   cd LiquidCpp
   ```

3. Run the build script:
   ```bash
   build.bat
   ```

### Setup Instructions

1. Run the setup script to download dependencies:
   ```bash
   setup.bat
   ```
